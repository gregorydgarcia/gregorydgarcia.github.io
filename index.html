RiverBatJS (Beta)

RiverBat is a Proof-of-concept web application that leverages the tools provided through MapBox to build a 100% in browser Geographic Information System. Visualization is handled with Mapbox's mapbox.js - an extended version of the fantastic open source mapping library Leaflet. Geoprocessing is handled by TurfJS - a (at times) client side spatial processing library also provided by Mapbox. For intactivity I used JQuery and the geodata management is crudely manipulated in a singleton until it is offloaded into the Leaflet object.

The goal of this project was to demonstrate that an affordable (see: free) GIS solution could be widely distributed through a browser. The app is simple enough to allow for people around the world to answer basic spatial questions and share spatial data.

Beer Points

Beer Points is a fun data visualization of the top 100 beers of five common american styles as voted on by the Beer Advocate community.

First I scrapped the beeradvocate site using python then converted the results into a geojson object. A few leaflet plugins latter and we had a pretty snazzy map.

Instead of hardcoding in the density by state, the values are calculated on the fly using TurfJS. I think spatially responsive websites will play a big part in the future.
